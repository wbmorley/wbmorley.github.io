Face Recogntion Project


This simple face recognition project is sectioned into two parts: face detection and face recognition. The first step is to run the face detection code which prompts the user to name found faces for the application to save and recognize. The second step, face recognition, allows the user to select a photo that the program will detect and caption with names which have been previosuly stored from step one.


View on Github

Face Detection


The application will first need to be told which name belongs to which face. The face detection step will find faces in a given picture and prompt for names.


The select photo pop-up will first appear and prompt for a photo selection. After selection, a pop-up to select a directory to store found faces will appear.


The next step is to put names to found faces. Each face will appear in a pop-up and prompt the user to enter a name.


Once the program has detected faces and the user has input corresponding names, the cropped face is saved as a .JPG file using it's respective name. The saved directories will be used in the face recognition step as potential matches for any faces put through the application.

Face Recognition


Once the application has detected faces and the user has given each face a corresponding name, the user can then run the recognition step and input any picture to scan for potential matches.

The user will select a photo or directory of photos to input through the application and the system will use previously saved faces as potential matches.
